### How to Setup & Run
1. Clone repository ini
2. Buka terminal di folder root repository ini
3. Jalankan perintah
```
uv sync
uv run src/ffnn.py
```
4. Untuk menjalankan notebook, pilih kernel dari venv yang sudah diaktifkan

Note: Pastikan versi Python >= 3.12
